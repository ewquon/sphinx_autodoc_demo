Getting Started¶

  1. Defaults in sphinx-quickstart are fine, just make sure you enable autodoc to “automatically insert docstrings from modules”.

  2. Edit conf.py to include in the header:

    import os
import sys
sys.path.insert(0, 'path/to/your/code')

  3. Add sphinx.ext.napoleon to the list of extensions to enable numpy-style docstrings (optional).

  4. Add to your rst file:

    .. automodule:: name_of_your_python_module
    :members:

    Note that sphinx must be able to import your python module from somewhere in the python system path.

  5. Run make html. When you update your python code, re-run make html.
